Automatic cricket highlights generation

May 2024

The project develops an automated system to generate concise cricket highlights by using image processing to remove redundant replays. It classifies video frames to identify key events like wickets and boundaries, ensuring engaging content. Advanced algorithms analyze visual and contextual information, enhancing the viewing experience for fans.

SOHO network- design and implementation

Nov 2023

The SOHO network project designs and implements an office network using VLANs, DHCP, and wireless access points. VLANs enhance security and performance by isolating traffic, while DHCP automates IP address assignment. Strategically placed access points ensure reliable connectivity, creating a seamless, scalable, and optimized network for growing office needs.

Secure text data transmission using Li-Fi

Jan 2023

The project on secure text data transmission using Li-Fi involves creating a Li-Fi prototype with Arduino. Using an LED and LDR with On-Off Keying modulation, the LED transmits data by switching on and off, and the LDR receives and converts it back into text. This demonstrates a secure, efficient, and cost-effective wireless communication method.
